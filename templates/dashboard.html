Refresh Token Guide

How to generate a new authentication token

  1. Log in to the Intuit OAuth Playground.
  2. Select your app from the dropdown and check the com.intuit.quickbooks.accounting scope.
  3. Click Get Authorization Code and log in to authorize access to your company.
  4. Click Get Tokens to generate a fresh Refresh Token.
  5. Copy the refresh token and paste it into the form on the settings page.

Pro Tip: Once established, the server will rotate these tokens automatically. You only need to do this manually if the connection expires or is revoked.
